# Express authentication

## Prerequisites

1. Install docker

  ```
  sudo curl -sSL https://get.docker.com/ | sh
  ```

2. [Install docker-compose](https://docs.docker.com/compose/install/)

## Getting started

* Production

  ```
  docker-compose up
  ```

* Development

  ```
  docker-compose -f docker-compose.yml -f compose-dev.yml up
  ```
